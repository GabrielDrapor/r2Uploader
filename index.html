Getting Started

Cloudflare Setup

You'll need your Cloudflare Account ID and an R2 API Token.

  1. Log in to Cloudflare Dashboard, find Account ID in URL.
  2. Go to R2 -> Manage API Tokens.
  3. Create a new API Token with "Admin Read & Write" permissions for your bucket.
  4. Copy the Access Key ID and Secret Access Key provided.

Your credentials are stored securely locally.
